# 原始需求記錄 (User Requirement Raw)

- **提出者**：使用者 / 2026-07-14
- **來源**：對話口述需求

## 原始描述

1. **財務分析範圍**：台股、美股，需涵蓋營收、EPS、毛利率、淨利率、負債比、現金流、本益比。
2. **股價預測目標與方法**：結合財報因子的統計模型，並加入時間序列模型；預測時間單位為「週」。
3. **資料來源**：公開資訊觀測站（MOPS）。

## 缺口拷問後補充確認（Gate 1 追加澄清）

4. **美股資料來源缺口**：MOPS 僅涵蓋台股上市櫃公司，無法取得美股資料。使用者確認美股財報改用 **SEC EDGAR（10-Q/10-K）**，股價與市場資料改用**市場資料 API**（如 Yahoo Finance / Alpha Vantage 類服務，實際供應商於 Phase 02/03 選型）。
5. **預測輸出目標**：未來一週**漲跌方向 + 幅度區間**（例如「上漲，幅度 2~5%」），非絕對股價數值、也非連續報酬率。
6. **系統呈現形式**：後端分析服務 + API，不含網頁儀表板（無需 Phase 02 UI 雛型）。
